# 部署方案

> 2026-07-23 · 打样快速报价(A家)
> 应用形态:纯静态网页(无后端、无数据库),构建产物仅 ~80KB,任何静态托管都能跑。

---

## 一、方案对比

| 方案 | 国内访问 | 成本 | 是否需要备案 | 评价 |
|---|---|---|---|---|
| GitHub Pages(现状) | 不稳定,部分网络慢/打不开 | 0 | 否 | 临时够用,不适合长期给客户用 |
| Cloudflare Pages | 一般(无国内节点) | 0 | 否(绑自有域名) | 比 GitHub 略好,仍不保证 |
| **腾讯云 CloudBase 静态托管(推荐)** | **快、稳定(国内 CDN)** | 域名 ~60元/年 + 流量费(本应用几乎为 0,每月几毛~几元) | **绑自定义域名需 ICP 备案**(免费,约 1-2 周) | 与微信小程序同生态,备案一次两头用 |
| 阿里云 OSS + CDN | 快、稳定 | 同上量级 | 需备案 | 同样好,但与微信生态无联动 |
| 轻量云服务器 + nginx | 快、稳定 | ~30-60元/月 | 需备案 | 目前是杀鸡用牛刀;等将来要自建后端再上 |

**结论:推荐腾讯云 CloudBase(云开发)静态托管。** 理由:①国内 CDN 加速,业务员和客户打开秒开;②后期微信小程序大概率用云开发放"云端规则配置"(改价不发版),现在就落在同一个环境里,一个账号一次备案全覆盖;③按量计费,本应用流量极小,近乎零成本。

## 二、分阶段实施

### 阶段 0:现状(已完成)

- GitHub Pages:https://leejunsc.github.io/dayang-baojia/
- 继续保留,作为备用链接和代码托管。

### 阶段 1:国内稳定访问(建议 1-2 周内完成,主要是等备案)

**需要您本人操作的**(涉及实名/支付,我无法代办):

1. 注册/登录腾讯云,完成个人或企业实名认证(建议企业认证,后期小程序也要企业主体)。
2. 购买域名(如 `xxxbz.com`,~60元/年;在腾讯云买,备案流程最顺)。
3. 提交 ICP 备案(腾讯云控制台引导,免费,个人约 1 周、企业 1-2 周,期间网站可先用默认域名调试)。
4. 开通"云开发 CloudBase"按量计费环境。

**我来完成的**:

5. 把 `dist/` 发布到 CloudBase 静态托管(CLI:`@cloudbase/cli`,一条命令)。
6. 备案通过后绑定自定义域名 + HTTPS 证书(免费证书,控制台一键)。
7. 改造 `scripts/deploy.sh`:一次构建,同时发布 GitHub Pages(备用)和 CloudBase(主用)。
8. (可选)配 GitHub Actions:推代码自动部署,手机上改完规则也能自动上线。

**产出**:`https://bj.您的域名.com` 这类正式链接,国内秒开,发客户名片、印宣传单都拿得出手。

### 阶段 2:微信小程序(域名备案完成后随时可启动)

1. 注册微信小程序(企业主体,认证费 300元/年)。
2. 用 uni-app 重建 UI 层,**报价引擎(src/engine)原样复用**,两端价格永远一致。
3. 规则配置上云(云开发数据库/云存储):改价在后台改,网页版和小程序同时生效,不用重新发版。
4. 小程序增值项:报价单分享图、客服会话入口、询价记录。
